# Capabilities

## Communication
- **Network diagnostics**: Ping, DNS resolution, port checking
- **Messaging**: Telegram, Discord, Slack integration
- **Email**: Basic read/send via OAuth (via 1Password integration)

## File Management
- **Read/Write**: Local files, config files, logs
- **Search**: Find files by name, extension, content
- **Compare**: Diff between versions
- **Version control**: Git add/commit/push management

## Execution
- **Shell**: Run bash commands, scripts
- **Process**: Start/stop background jobs, monitor CPU/memory
- **Service**: Restart/stop/openclaw-gateway via systemd
- **Network**: Port scanning, socket checks

## AI & Data
- **Memory**: Retrieve from MEMORY.md and memory/*.md
- **Summarization**: Condense long texts
- **Translation**: Language translation
- **Analysis**: Pattern recognition, data extraction

## Infrastructure
- **Disk**: Usage monitoring (df, du)
- **Storage**: File integrity checks
- **Time**: System clock, uptime tracking
- **Locks**: File locking for concurrency safety

## Automation
- **Cron**: Schedule tasks, monitor events
- **Heartbeat**: Periodic checks, status acks
- **Task queue**: Manage pending operations

## Security
- **Auth**: Token management, credential storage
- **Whitelist**: Command execution allowlist
- **Audit**: Log execution history

## Integration
- **Browser**: Web navigation, scraping, snapshot
- **Files**: Mount/access external storage
- **Media**: Image display, audio transcription (Whisper)
